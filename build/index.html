Прозрачно и Просто
Ethereum Smart Contract Для розыгрыша используется smart contract в сети Ethereum
Честное определение победителя Алгорит определения победителя, разделен на два независимых источника случайных данных (для генерации ключа который взаимодействует только в момент розыгрыша), что повышает стохастичность процесса
Прозрачно и безопасно Исходный код в открытом доступе на Github и открытая история розыгрышей доступна в сети Ethereum
Ставка в любой криптовалюте Для игры можно использовать Bitcoin, Ethereum, Litecoin, Ripple, Bitcoin Cash. Список будет постоянно обновляться
Как это работает?
Вначале каждого раунда участники делают ставки, они покупают виртуальные билеты, один из которых станет победным. Номер последнего билета, передается в смарт-контракт Ethereum сразу после завершения ставок в раунде, где и просходит определение выигрышного билета. Для шифрования используется наиболее криптостойкий алгоритм Keccak (SHA-3)
